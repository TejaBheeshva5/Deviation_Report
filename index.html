Load from Google Sheets (via backend)

Uses a small Apps Script backend — your Google account, not the viewer's, reads the sheets
This dashboard talks to one backend URL (an Apps Script Web App you deploy once) instead of Google Sheets directly. The backend reads System Revenue, Video Revenue, and DRR Revenue using your own access and hands back plain JSON — so people opening this dashboard never need Google Sheets access or sign-in at all. See SETUP.md for the one-time deployment steps.
